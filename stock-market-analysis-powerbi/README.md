# 📊 Stock Market Data Analysis using Power BI

## 📌 Project Overview
This project focuses on analyzing real-time stock market data using Microsoft Power BI. The dashboard fetches live data from Yahoo Finance API and transforms it into meaningful insights using interactive visualizations.

It helps users understand stock performance, compare companies, and make informed decisions through data-driven analysis.

---

## 🎯 Objectives
- Fetch real-time stock data using Yahoo Finance API
- Clean and transform data using Power Query
- Build interactive dashboards in Power BI
- Analyze stock trends and performance

---

## 🛠️ Tech Stack
- Microsoft Power BI
- Power Query
- Yahoo Finance API
- Data Visualization

---

## 📊 Features
- 📡 Real-time stock data (Price, Volume, Market Cap)
- 📈 Interactive charts (Line, Bar, Area, Donut)
- 🔍 Filters and slicers for dynamic analysis
- 🆚 Multi-stock comparison
- 📊 KPI cards for quick insights

---

## 🧠 Methodology
1. Data Collection from Yahoo Finance API
2. Data Cleaning using Power Query
3. Data Transformation and Modeling
4. Dashboard Creation
5. Analysis and Insights
