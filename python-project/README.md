# 🐍 Python Project - Driver Drowsiness Detection

## 📌 Description

This project detects driver drowsiness using a webcam.
It monitors eye movement and alerts the user when eyes remain closed.

## 🚀 Features

* Face detection using OpenCV
* Eye detection
* Detects closed eyes
* Alert using beep sound

## 🛠 Tech Used

* Python
* OpenCV
* NumPy

## ⚙️ How It Works

* Webcam captures video
* Face and eyes are detected
* If eyes are closed for many frames → alert is triggered

## 📂 Files

* main.py
* haarcascade files

## ▶️ Run

```bash
pip install opencv-python numpy
python main.py
```
